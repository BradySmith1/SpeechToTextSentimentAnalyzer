Authors: Brady Smith, Andrew Monroe, Nolan Flinchum
Version: 4/28/2023

Description:
    This program takes in the user's words using speech-to-text, analyzes the sentiment
    of what they say, and outputs a text-to-speech response based on the sentiment
    of the statement. Through use of this program, hopefully users will be able to speak
    with kind words instead of negative ones.

Commands to install Required Libraries:
    pip install pyttsx3
    pip install playsound
    pip install tkinter
    pip install sounddevice
    pip install pydub
    pip install --upgrade google-cloud-language
